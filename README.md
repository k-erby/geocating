## Installation

This uses nestJS (express) for the arch and prisma (sqlite) for the data management. To run the server, start it up, then go to http://localhost:3000/.

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
